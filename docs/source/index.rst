tiltr
=====

**tiltr** is a Python package for visualizing and exploring the spin-orbit
architecture of exoplanetary systems using the TEPCat obliquity catalogue.

Features
--------

* Plot projected obliquity (λ) versus stellar parameters.
* Plot true obliquity (ψ) distributions.
* Highlight individual planetary systems.
* Access the TEPCat obliquity catalogue programmatically.
